設定

⚠ このアプリはブラウザから直接 Claude API を呼び出します。API キーはこの端末の localStorage にのみ保存されますが、共有端末では使用しないでください。

Anthropic API キー(sk-ant-...)を入力してください。

内規(社内ルール) — チーム共通の1つを運用します。回答時にAIが参照し、共有ONの場合はチーム全員に反映されます。

☁ チーム共有(任意)

Firebase を使うと、内規と質問⇄回答を複数の担当者・端末で共有できます。設定するには下の2つを入力してください(未設定ならこの端末のみで動作)。現在の状態: ローカルのみ

① Firebase 設定(コンソールの firebaseConfig の { ... } を貼り付け)

② ワークスペース合言葉(チームで同じ文字列を共有。これを知る人だけが同じデータにアクセスできます)

共有ONの場合、内規と質問⇄回答は Firebase(クラウド)に保存され、ログインした担当者のみアクセスできます。合言葉は共有パスワードとして厳重に管理してください。セットアップ手順は README を参照。

担当者ログイン

チーム共有を利用するにはログインが必要です。管理者に発行されたアカウントでログインしてください。
