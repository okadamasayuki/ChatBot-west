# 会計相談チャット

LINE風のチャットUIで会計に関する質問ができるWebアプリです。Claude API が質問に回答し、AIで回答できない質問は会計の専門家にエスカレーションされます。

## 機能

- 💬 **LINE風チャットUI** — 質問者はチャット形式で会計の質問を送信
- 🤖 **Claude API による自動回答** — 一般的な会計知識で答えられる質問はAIが即座に回答
- ❓ **不足情報の聞き返し** — 情報が足りず正確に回答できない場合は、AIが「どんな情報が必要か」を具体的に列挙して質問者に聞き返す
- 👤 **専門家エスカレーション** — AIが回答できない質問(個別の税務判断が必要な場合など)は、AIが回答方針の選択肢を生成して専門家タブに転送
- ✍️ **回答文(案)の自動生成** — 専門家が選択肢を選ぶと、Claude が回答文案を作成。専門家は編集のうえ「OK」ボタンで質問者に送信
- ⬇️ **Q&A履歴のダウンロード** — 質問と回答のペアを JSON / CSV(Excel対応・BOM付きUTF-8)でダウンロード可能

## 使い方

1. GitHub Pages のURLを開く
2. 初回起動時に Anthropic API キー(`sk-ant-...`)を入力
   - キーは [Anthropic Console](https://platform.claude.com/) で取得できます
   - キーはブラウザの localStorage にのみ保存されます
3. 「質問者」タブで会計の質問を入力
4. AIが回答できない質問は「専門家」タブに届きます(バッジで件数表示)
5. 専門家は回答方針を選択 →「回答文(案)を作成」→ 内容を確認・編集 →「OK — 質問者に送信」
6. ヘッダーの「⬇ 履歴」からQ&A履歴をダウンロード

## アーキテクチャ

- 完全に静的な単一HTMLファイル(`index.html`)— ビルド不要、GitHub Pages でそのままホスティング
- ブラウザから Claude API(`claude-opus-4-8`)を直接呼び出し(CORS対応ヘッダー使用)
- 質問の振り分けには構造化出力(JSON Schema)を使用し、「回答」か「エスカレーション+選択肢生成」かをAIが判断
- データ(チャット履歴・Q&A履歴・APIキー)はすべてブラウザの localStorage に保存

## 注意事項

- ⚠️ APIキーはブラウザに保存されるため、**共有端末では使用しないでください**
- このデモでは質問者ビューと専門家ビューが同一ブラウザ内のタブ切替になっています(サーバーレス構成のため)。実運用でユーザー間のリアルタイム通信が必要な場合は、バックエンド(例: Firebase, Supabase)の追加が必要です
- AIの回答は一般的な情報提供であり、税理士・公認会計士による助言に代わるものではありません

## デプロイ

`claude/accounting-chat-app-k99ti9` ブランチ(または `main`)への push で GitHub Actions が自動的に GitHub Pages へデプロイします(`.github/workflows/deploy-pages.yml`)。
